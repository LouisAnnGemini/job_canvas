# TaskFlow - 功能说明书

TaskFlow 是一款功能全面、灵活且注重视觉组织的任务管理应用。它提供了结构化的表格视图和自由形式的白板视图，允许用户以最适合其工作流程的方式管理任务。

## 核心功能

### 1. 双重视图模式
*   **表格视图 (Table View)：** 一个高度可定制的数据网格，用于结构化的任务管理。
*   **白板视图 (Whiteboard View)：** 一个无限画布，用于视觉组织、空间分组和思维导图。

### 2. 任务管理
*   **创建、编辑、删除：** 完整的任务增删改查功能。
*   **复制任务：** 快速克隆现有任务。
*   **任务详情弹窗：** 全面查看所有任务属性，包括在表格中隐藏的字段。
*   **行内编辑 (表格视图)：** 直接在表格单元格内编辑任务字段。
*   **批量编辑：** 选择多个任务并同时更新它们的一个特定字段。
*   **任务关联：** 在相关任务之间创建双向链接。关联的任务会显示在任务详情弹窗中，方便快速导航。在白板视图中，关联的任务之间会用带箭头的有向连线连接。
*   **拖拽排序：** 在表格视图中，可以通过拖拽每一行左侧的抓取手柄来重新排序任务。

### 3. 自定义列 (表格视图)
*   **动态列管理：** 添加、重命名、删除和重新排序自定义列。
*   **丰富的列类型：** 支持文本 (Text)、数字 (Number)、日期 (Date)、复选框 (Checkbox)、单选 (Single Select) 和多选 (Multi-select)。
*   **自定义选项与颜色：** 对于单选和多选列，用户可以定义自定义选项并为它们分配特定的颜色（例如，紧急程度级别）。
*   **列宽调整：** 通过拖动列边界手动调整列宽。
*   **可见性切换：** 隐藏或显示特定的列，以便专注于相关数据。
*   **文本换行：** 切换单元格内的文本换行，以适应长描述。
*   **圆点多选框：** 表格视图中用于选择任务的复选框（包括全选和单行选择）采用了圆点样式，提供类似单选按钮的精致视觉效果，同时保留了多选的交互功能。

### 4. 数据持久化与导入导出
*   **本地存储 (Local Storage)：** 所有数据（任务、列、框架、模板）都会实时自动保存到浏览器的 `localStorage` 中，确保会话之间不会丢失数据。
*   **导出为 Excel：** 支持将选中的任务导出为 Excel (`.xlsx`) 文件。如果未选择任何任务，则默认导出当前过滤后的所有任务。
*   **JSON 导入/导出：** 将整个工作区状态导出为 JSON 文件并在以后导入，方便进行数据备份和共享。

### 5. 白板交互
*   **无限画布：** 在工作区内自由平移和缩放。
*   **空间组织：** 拖放任务卡片以进行视觉排列。
*   **框架 (Frames)：** 创建框架以将相关任务分组在一起。移动框架时，框架内的任务会一起移动。
*   **对齐工具：** 选择多个任务并在水平（`H` 键）或垂直（`V` 键）方向上以紧凑的间距对齐它们。
*   **有向连线：** 在画布上，关联的任务之间会自动用带箭头的连线连接，清晰展示任务间的流向或依赖关系。

### 6. 搜索与过滤
*   **全局搜索：** 在任务标题和描述中搜索关键字。
*   **高级过滤：** 基于单选和多选列中的特定选项过滤任务。
*   **实时更新：** 搜索和过滤条件会立即更新表格和白板视图。

### 7. 任务模板
*   **模板管理：** 创建、重命名、复制和删除可重用的任务模板。一个模板可以包含一个或多个预设任务。
*   **默认值设置：** 可以在模板中为包含的任务预先填写字段值。
*   **使用模板：** 基于模板快速生成新任务。在正式添加任务之前，用户可以在弹出的表单中修改或补充特定细节，而不会改变原始模板的结构。

## 键盘快捷键 (白板视图)
*   `H`：水平对齐选中的任务。
*   `V`：垂直对齐选中的任务。
*   `F`：在选中的任务周围创建一个框架。
*   `Delete` / `Backspace`：删除选中的任务或框架。
*   `Ctrl+C` / `Cmd+C`：复制选中的任务。
*   `Ctrl+V` / `Cmd+V`：粘贴复制的任务。
*   `Space` (按住)：启用平移模式。
